# Swoole Net Utils

[[English Readme](README.md)]  [[中文说明](README_CN.md)]

Swoole Net Utils is a set of net utils based on php swoole

It contains:
* swoole_http_proxy --- a http proxy server
* swoole_port_forwarding --- a port forwarder
